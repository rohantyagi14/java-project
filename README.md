# Console Quiz App

A simple console-based Quiz Application developed using Java. 
This project is created to practice Java fundamentals and Object-Oriented Programming (OOP) concepts.

## Features
- Multiple choice questions (MCQ)
- User input handling
- Score calculation
- Instant answer checking
- Console-based user interface

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Scanner Class

## OOP Concepts Used
- Classes and Objects
- Encapsulation
- Methods
- Constructors

## How to Run
1. Clone this repository
2. Open the project in any Java IDE (IntelliJ IDEA / Eclipse)
3. Run the main class
4. Play the quiz in the console

## Project Purpose
This project was built to improve my understanding of Java programming and OOP concepts through practical implementation.
