# Console Quiz App

A console-based Quiz Application developed using Java. 
This project is created to practice Java fundamentals and Object-Oriented Programming (OOP) concepts.

## Features

- Multiple choice questions (MCQ)
- Console-based user interaction
- Stores questions using Java objects
- Checks user answers
- Calculates final score

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Scanner Class

## OOP Concepts Used

- Classes and Objects
- Encapsulation (Private variables with getters and setters)
- Constructors
- Methods
- Array of Objects

## Project Structure

Console-Quiz-App
│
├── Main.java
│   - Application starting point
│
├── Question.java
│   - Stores question details and options
│
├── QuestionService.java
│   - Handles quiz logic and score calculation
│
└── README.md

## How to Run

1. Clone this repository
2. Open the project in IntelliJ IDEA or any Java IDE
3. Run the Main.java file
4. Answer the questions in the console

## Project Purpose

This project was built to improve my understanding of Java programming and Object-Oriented Programming concepts by implementing a real-world console application.
